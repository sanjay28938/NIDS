## Dataset Information

The `country_asn.csv` dataset is not included in this repository due to its large file size and GitHub storage limitations.

To run this project successfully, download the dataset separately and place it in the following directory:

```text
data/country_asn.csv
```

Please ensure the file name and directory structure remain unchanged, as the application expects the dataset at this location.

## Model File

The trained model file `ids_model.pkl` is not included in this repository due to file size limitations.

To run the project, place the trained model in the following directory:

```text
models/ids_model.pkl
```

Alternatively, you can generate the model locally by running the training script:

```bash
python train_model.py
```
